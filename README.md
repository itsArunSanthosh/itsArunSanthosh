### Feature Project

## AI-Powered Network Intrusion Detection System (NIDS)

A production-style system designed to detect malicious network activity in real-time using AI & ML models.

## What it does:
- Detects anomalies and attacks in network traffic
- Combines multiple ML models for robust prediction
- Uses Graph Neural Networks (GNNs) to capture multi-hop relationships
- Generates actionable alerts for security systems (SIEM/SOC)

## Key contributions:

- Designed multi-model ensemble system with meta-learner
- Implemented feature engineering pipelines for network flow data
- Built real-time inference pipeline with Kafka + FastAPI
- Solved training-serving skew using feature store (Feast)
- Developed model explainability pipeline for GNN predictions
- Integrated monitoring & alerting system for production reliability
